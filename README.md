## Project Description
This is a web-based simulation of a Scalable Thread Management Library. It demonstrates:
* Thread Creation and Lifecycle Management (Ready, Running, Terminated).
* Round Robin Scheduling Algorithm.
* Synchronization using Mutex locks to prevent race conditions.
* Real-time visualization of thread states.

## How to Run
1. Open `index.html` in any modern web browser.
2. Click "Initialize Simulator".
